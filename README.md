```
$ make
```

## Como usar

```
$ cesar -h

---- Instituto Federal de Rondônia - Campus Ariquemes ----
Criptografa e descriptografa mensagens usando Cifra de César
N. A. V. Simões <natanael.simoes@ifro.edu.br>
http://github.com/ifroariquemes/CifraCesar

Uso:

      cesar -c|-d mensagem -k chave

 -c mensagem      Criptografa uma mensagem
 -d mensagem      Decriptografa uma mensagem
 -k chave         Aplica uma chave na de/criptografia
 -h               Mostra essa mensagem
 Criptografar: cesar -c mensagem -k 3
 Decriptografar: cesar -d phqvdjhp -k 3
 ```
